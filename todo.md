# Data table

## **Toolbar**

- Global search
- Search column settings (choose searchable fields)
- Breadcrumbs / chips for applied filters, sorts, and active search
- Clear-all-filters button
- Export actions (CSV / XLSX / JSON / Print)
- Bulk actions bar (appears when rows selected)
- Density selector (compact / normal / comfortable row spacing)
- Refresh button

---

## **Table Header**

### **Column Features**

- Default column header
- Select-all rows checkbox
- Sorting support
  - Single-column sorting
  - Multi-column sorting (Shift+click)
  - Sort indicators (asc/desc/remove)
- Filtering support
  - Basic search input filter
  - Filter button with custom input
  - Enum/multi-enum filter
    - AND vs OR logic mode
  - Date filter (date or date range)
  - Number range filter
  - Text match options: contains / equals / starts with / ends with / regex
  - Filter indicators
- Column customization
  - Resizable columns
  - Draggable column reordering
  - Column visibility panel
  - Column width auto-adjust
  - Column lock/freeze (sticky left/right)
- Sticky header
- Sticky first column
- Column-level tooltips/help icons
- Date Range Selector

---

## **Table Body**

### **Rows & Cells**

- Custom cell rendering per column
- Row selection (checkbox, click-to-select)
- Click-to-focus cells
- Cell tooltips (for truncated or rich data)
- Conditional row or cell styling (status/highlight)

### **State & UX**

- Virtualized rendering (for thousands of rows)
- Row hover effects
- Zebra striping
- Loading state (spinners or skeleton rows)
- Empty state (“No results found”)
- Loading skeleton
- Indeterminate checkbox state (for selected + partially selected)

---

## **Pagination**

- Total row count
- Selected row count
- Rows per page selector (with presets)
- Current page indicator
- Next page
- Previous page
- Jump to page input
- Go to first / last page buttons
- Server-side pagination mode (optional)
- Auto-scroll to top on page change
